# Student Grade Tracker

## Project Overview

Student Grade Tracker is a Java-based console application developed to manage and analyze student academic performance. The system allows users to enter student names and marks, calculate statistics, and generate a summary report.

## Features

* Add student details
* Store student marks using ArrayList
* Calculate average marks
* Find highest marks
* Find lowest marks
* Display complete student report

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Project Structure

* Student.java
* GradeTracker.java

## How to Run

1. Open the project in VS Code or any Java IDE.
2. Compile the Java files:
   javac Student.java GradeTracker.java
3. Run the application:
   java GradeTracker
